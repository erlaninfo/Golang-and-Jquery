## 🎓 Instalação

Você precisa ter a biblioteca postgres


```sh
$ go get github.com/lib/pq
```

Em seguida modifique a string de conexão em: goweb/db.go 

Você precisa modificar os carminhos, de acordo onde sua pasta está. github.com/goweb
